# PRD: 竞品早期营销策略溯源研究

## Overview

深度研究 AI Coding 领域 6 个竞品的早期营销策略，溯源流量暴涨的关键节点，为 Trellis 制定首发策略提供数据支撑。

## Background

Trellis 即将开源发布，需要了解：
1. 同类产品是如何从 0 到 1 积累用户的
2. 哪些营销渠道/策略最有效
3. 如何复制成功模式

### Key Findings

#### 6 种早期营销模式

1. **YC 系统打法** (Continue) - YC → HN → 媒体
2. **品牌借势 + 播客** (OpenCode) - 已有品牌 → HN → 播客
3. **博客首发 + KOL** (Superpowers) ⭐ - 方法论 → KOL 放大
4. **概念顺风车** (OpenSpec) - 抢占概念 → 等待验证
5. **内部工具 + YouTube** (Roo Code) - 开源 → 教程矩阵
6. **纯产品驱动** (claude-mem) - 强痛点 → 被动发现

#### 核心真相

- **KOL 推荐 > 渠道曝光** - 一个高质量 KOL > 100 条推广
- **方法论内容 > 产品公告** - 教育市场同时建立权威
- **时机窗口 > 一切** - 借势大厂发布
- **创始人品牌是长期资产**
- **YouTube 是被低估的增长渠道**

### Trellis 建议

**推荐策略**: Superpowers 模式（博客首发 + KOL 放大）

**行动优先级**:
1. 🔴 找到你的 Simon Willison（KOL 关系建立）
2. 🔴 方法论博客先行（建立权威）
3. 🔴 概念绑定（品类占领）
4. 🟡 HN Show/Launch
5. 🟡 创始人品牌建设
6. 🟢 播客/采访、内容矩阵、YouTube 生态
